## How Can I Contribute?

### Reporting Bugs

If you find a bug, please create a new issue with the **Bug Report** template and include:

- A clear and descriptive title
- Steps to reproduce the issue
- Expected behavior vs actual behavior
- Screenshots or screen recordings (if applicable)
- Your environment (OS, version, etc.)

### Suggesting Features / Enhancements

Feature requests are welcome! Please open an issue using the **Feature Request** template and include:

- A detailed description of the proposed feature
- Why it would be useful to the project
- Any examples or references

### Improving Documentation

Documentation improvements are always valuable. Feel free to:

- Fix typos
- Clarify confusing sections
- Add examples
- Improve README or other docs
